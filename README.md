# Trabalho-Final-Algoritmos-E-Programa-o-UFRGS-Plisnos

Jogo plataforma "estilo Donkey Kong" (mais ou menos).
Possui high score individual por level persistente a cada playthrough assim como continue, que começa a partir do ultimo level jogado.
O jogador possui 3 vidas e perde elas a cada hit.
Para matar inimigos o jogador deve pular em cima deles, como no mario clássico.
O jogo possui menu e level select assim como um sistema de unlock dos níveis.
O jogo possui pulo e método de escalar escadas (ao invés de teleporte puro).
O jogo está pré-compilado para linux mas como a raylib é multiplataforma, recompilar para windows é extremamente simples.
